# GroceriesHelper

GroceriesHelper is a simple command-line tool to help me quickly calculate my grocery spending.

## How to Download

You can download the `main.py` file using a command-line tool like `curl` or `wget`.

### Using curl

```bash
curl -o main.py https://raw.githubusercontent.com/dielsonsales/GroceriesHelper/refs/heads/main/main.py
```

### Using wget

```bash
wget -O main.py https://raw.githubusercontent.com/dielsonsales/GroceriesHelper/refs/heads/main/main.py
```

## How to run

Once downloaded, you can run the script using:

```bash
python3 main.py
```
